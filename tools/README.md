# 🛠️ Инструменты бота

## 📁 Содержимое папки

### logger.py
- **Основной модуль логирования** для бота
- **5 типов логов:** основные, ошибки, системные, Discord, Ollama
- **Автоматическое создание** папки logs
- **Детальная информация** с временными метками

### view_logs.py
- **Интерактивный просмотр** логов бота
- **Фильтрация по типам** (ошибки, Discord, Ollama)
- **Поиск по тексту**
- **Показ последних N строк**

### clean_logs.py
- **Очистка старых логов** (более 7/30 дней)
- **Статистика использования**
- **Расчет освобожденного места**

## 🚀 Использование

```bash
# Просмотр логов
python tools/view_logs.py

# Очистка логов
python tools/clean_logs.py
```

## 📊 Структура логов

```
logs/
├── bot_2025-01-17.txt      # Основные логи
├── errors_2025-01-17.txt   # Ошибки
├── system_2025-01-17.txt   # Системные события
├── discord_2025-01-17.txt  # Discord события
└── ollama_2025-01-17.txt   # Ollama запросы
```

## 💡 Особенности

- **Автоматическое создание** папки logs
- **Ежедневная ротация** логов
- **Кодировка UTF-8** для русских символов
- **Детальная диагностика** всех процессов
